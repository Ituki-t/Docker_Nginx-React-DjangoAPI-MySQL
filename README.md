# Docker_nginx-react-djangoapi-
### 概要
- dockerでNginx, React, Django, MySQLコンテナを立ててNginxでReactの画面を表示させる

### 場所
```
cd projects\docker\docker_nginx-react-djangoapi-mysql
```

### exec
- app
```
docker exec -it app bash
```
- api
```
docker exec -it api bash
```
